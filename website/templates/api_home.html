This API will be able to searcha and display data from both 2017 and 2018 NCAA championship matches. Instructions on how to use are below.

NCAA API Team and Player Search



Available API Routes:


/api/v1.0/team_names
- Display all of the team names for easy reference to search.

/api/v1.0/team
- Display all teams and available stats.

/api/v1.0/team/(name)
- Search for one particular team. Example: /api/v1.0/team/Florida%20St

/api/v1.0/player_names
- Display all player names for easy reference to search.
