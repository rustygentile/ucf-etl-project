NCAA API Team and Player Search

This API will be able to search and display data from both 2017 and 2018 NCAA championship matches. Instructions on how to use are below.

Available API Routes:


2017 All Team Names

- Display all of the team names from NCAA 2017 (easy reference to search)

Example: /api/2017/team_names

2017 All Team Names Info!

- Display all teams and available stats: from NCAA 2017

Example: /api/2017/team

2017 Select the Team by Name

- Search for one particular team.

Example: /api/v1.0/team/Florida%20St

2017 All Player Names

- Display all of the player names from NCAA 2017 (easy reference to search)

Example: /api/2017/player_names

2017 All Player Names Info!

- Display all Player and available stats: from NCAA 2017

Example: /api/2017/player

2017 Select the Player Name

- Search for one particular player.

Example: /api/v1.0/player/Nick%20DiNardi

2018 Elite Eight Player Names

- Display all of the player names from the NCAA 2018 Elite Eight (easy reference to search)

Example: /api/2018/elite_player_names

2018 Elite Eight Players

- Display all Elite Eight players and available stats

Example: /api/2018/elite_eight

2018 Select the Elite Player Name

- Search for one particular player.

Example: /api/v1.0/player/Nigel%20Hayes

2018 Championship Game

- Display the 2018 Championship Final

Example: /api/2018/championship

2018 Select the region Name

- Search for one particular region Teams with seedings.

Example: /api/2018/Seeds_region/EAST
